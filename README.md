# AdathinkLab_Promises

## Reto Práctico N°1

1ra clase de Node.js - Bootcamp Adathink

### Uso CLI

Antes de ejecutar el programa, instale las dependencias:

`$ npm install`

#### Ejecutar de la siguiente manera a través de la terminal:

  ```
      $ pokemon <id or name>
```
